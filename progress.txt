## Codebase Patterns
- Main page is a single index.html with inline styles and one JS file (js/bitcoinprice.js)
- Price fetched from Coinbase API: https://api.coinbase.com/v2/prices/{COIN}-USD/spot
- Price display element id: btc-price-display, container id: btc-container
- `currentCoin` variable in js/bitcoinprice.js controls which coin is fetched/displayed
- Use `switchCoin(code)` to change coins - handles localStorage persistence + price refresh
- Main update function is `updatePrice()` - call it after changing `currentCoin` to refresh
- No build tools, linting, or tests - this is a plain HTML/JS project
- Search input: #coin-search (container div), #coin-input (text input), toggle via .visible class
- `showSearch()` and `hideSearch()` functions control search input visibility
